Google APIs Client Library for Java

Written by Google, this library is an easy-to-use and efficient Java client library for accessing Google APIs using JSON and OAuth 2.0.

Works on these platforms that support the Java language:

  • Java 5 (or higher) standard (SE) and enterprise (EE)
  • Android
  • Google App Engine

To start, please read the Developer's Guide.

Next, download the Google API libraries and try the samples.

Read the Get Help page to ask questions, report a bug, or subscribe to announcements.

This library is built on top of two common libraries, also built by Google, but designed to work with any HTTP service on the web: the Google HTTP Client Library for Java and the Google OAuth Client Library for Java.

Status: This library is in beta and under active development.
